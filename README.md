# Bluestock Mutual Fund Analysis

## Overview

An end-to-end mutual fund analytics project built using Python, SQLite, and Power BI.

## Features

* ETL Pipeline
* Data Cleaning
* Exploratory Data Analysis
* Performance Analytics
* Risk Analytics
* Power BI Dashboard
* Fund Recommendation Engine

## Tech Stack

* Python
* Pandas
* NumPy
* SQLite
* Matplotlib
* Seaborn
* Plotly
* Power BI

## Project Structure

data/
notebooks/
reports/
scripts/
sql/

## Setup

### Clone Repository

git clone <repository-url>

### Install Requirements

pip install -r requirements.txt

### Run Pipeline

python scripts/run_pipeline.py
